Explain the concept of responsive Design. what does the term mean.

ANS: It is a graphic user interface design that is developed in order to create content that adjust smoothly along with screen size In simple words it is a type of design which makes user to load up the page on different size of display by getting adaptable with screen resolution size. it generally focused on cascading style sheet(CSS) with different properties that makes website more professional.
